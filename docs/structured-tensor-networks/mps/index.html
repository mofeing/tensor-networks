        Matrix Product States #

        Origins: DMRG #

        The Density Matrix Renormalization Group algorithm is a numerical method developed by Stephen White

        Infinite DMRG #

        Step 1: Superblock Hamiltonian #

        \[H = - J \sum_{j=1}^N \sigma_j \sigma_{j+1} - h \sum_{j=1}^N \sigma_j\]

        Step 2: Select target state #

        \[H_{BB} = Q \Lambda Q^\dagger\]

        Usually, the desired state is the ground state which corresponds to eigenstate with the lowest eigenvalue. \[\Psi = Q(\min \lambda)\]

        Step 3: Reduced density matrix #

        \[\rho_{i j;i^\prime j^\prime} = \sum_{k,w} \Psi_{i,j,k,w} \Psi^*_{i^\prime,j^\prime,k,w}\] \[\tilde\Psi_{ij}= \text{Tr}_{k,w}[\Psi_{ijkw}] % \to \rho = | \tilde\Psi \rangle \langle \tilde\Psi |\] \[\rho_{i j;i^\prime j^\prime} = \sum_{k,w} (\text{Tr}_{ij}[\Psi_{ijkw}])^2 |\tilde\Psi_{ij} \rangle \langle \tilde\Psi_{ij} |\]

        Step 4: Low-rank representation of the reduced density matrix #

        \[\rho = U \Lambda U^\dagger\] \[\Lambda_{ij} = \lambda_i \delta_{ij}\] \[\Lambda_{ij} = \begin{cases} \lambda_i &\text{if } i = j \\ 0 &\text{otherwise} \end{cases}\] \[\tilde\Lambda_{ij} = \begin{cases} \lambda_i &\text{if } i = j \land i \leq m \\ 0 &\text{otherwise} \end{cases}\] \[H_B = \tilde{U}_{ij} = U_{ij}\]
